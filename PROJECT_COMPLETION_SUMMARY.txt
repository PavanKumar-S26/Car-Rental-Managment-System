### Quick Start (5 minutes)

```bash
# 1. Clone repo
git clone https://github.com/PavanKumar-S26/Car-Rental-Managment-System.git

# 2. Create namespace
kubectl create namespace carrental

# 3. Deploy with Helm

```

### With Custom Images

```bash

  --namespace carrental \
  --set backend.image.repository=myregistry/carrentalbackend \
  --set backend.image.tag=v1.0 \
  --set frontend.image.repository=myregistry/carrentalfrontend \
  --set frontend.image.tag=v1.0
```
